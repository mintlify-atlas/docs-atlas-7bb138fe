The Shopify Subscriptions Reference App includes five UI extensions that extend subscription functionality across Shopify's ecosystem. Each extension targets specific surfaces to provide seamless subscription management.

## Extension Architecture

Extensions in this app use Shopify's UI Extensions framework to render custom interfaces at specific extension points. They follow a modular architecture with:

- **Extension Targets**: Specific surfaces where the extension renders
- **API Access**: GraphQL and REST API capabilities
- **Network Access**: Ability to communicate with external services
- **Localization**: Multi-language support via i18n

## Extension Configuration

All extensions are configured via `shopify.extension.toml` files in their respective directories:

```bash
extensions/
├── buyer-subscriptions/
│   └── shopify.extension.toml
├── admin-subs-action/
│   └── shopify.extension.toml
├── thank-you-page/
│   └── shopify.extension.toml
└── pos-extension/
    └── shopify.extension.toml
```

## Common Patterns

### API Integration

Extensions use a consistent pattern for API calls:

```typescript
import {useExtensionApi} from 'foundation/Api';

function MyComponent() {
  const {i18n, graphql} = useExtensionApi();
  
  // Use graphql for queries and mutations
  const {data, loading, error} = useQuery(MY_QUERY);
}
```

### Error Handling

All extensions implement error boundaries and loading states:

```tsx
<ErrorBoundary>
  {loading ? <LoadingState /> : <Content />}
</ErrorBoundary>
```

### Internationalization

Extensions support multiple languages using Shopify's i18n utilities:

```typescript
const {i18n} = useExtensionApi();
const title = i18n.translate('subscriptions');
```

```bash
npm run dev
```

### Testing

Each extension includes comprehensive test coverage:

```bash
npm run test
```

### Deployment

Deploy extensions with:

```bash
npm run deploy
```
