The Buyer Subscriptions extension provides a complete subscription management interface within the customer account. Customers can view, pause, resume, cancel, and modify their subscriptions.

## Extension Configuration

The extension is configured in `shopify.extension.toml`:

```toml
api_version = "2025-01"

[[extensions]]
type = "ui_extension"
name = "Management page"
handle = "buyer-subscriptions"

  [[extensions.targeting]]
  module = "./src/index.tsx"
  target = "customer-account.page.render"

  [extensions.capabilities]
  api_access = true
  network_access = true
```

### Extension Target

- **Target**: `customer-account.page.render`
- **Surface**: Customer account pages
- **Capabilities**: API access, network access

## Architecture

The extension uses a router-based architecture to handle multiple views:

```tsx
import {reactExtension} from '@shopify/ui-extensions-react/customer-account';
import {Router} from './App';
import {ErrorBoundary} from 'foundation/ErrorBoundary';

export default reactExtension('customer-account.page.render', () => <App />);

function App() {
  return (
    <ErrorBoundary>
      <Router />
    </ErrorBoundary>
  );
}
```

## Key Features

### Subscription List View

Displays all customer subscriptions in a grid layout:

```tsx
import {
  Grid,
  BlockStack,
  Page,
} from '@shopify/ui-extensions-react/customer-account';
import {useSubscriptionListData} from './hooks/useSubscriptionListData';

export function SubscriptionList() {
  const {i18n} = useExtensionApi();
  const {data, loading, error, refetchSubscriptionListData} =
    useSubscriptionListData();

  if (loading && !data) {
    return <SubscriptionListLoadingState />;
  }

  if (data?.subscriptionContracts.length === 0) {
    return <SubscriptionListEmptyState />;
  }

  return (
    <Page title={i18n.translate('subscriptions')}>
      <Grid
        columns={['fill', 'fill', 'fill']}
        spacing="loose"
        rows="auto"
      >
        {data?.subscriptionContracts.map((contract) => (
          <SubscriptionListItem key={contract.id} {...contract} />
        ))}
      </Grid>
    </Page>
  );
}
```

### Subscription Details View

Provides detailed information and management actions:

```tsx
export function SubscriptionDetails({id}: {id: string}) {
  const {data, loading, error, refetchSubscriptionContract} =
    useSubscriptionContract({id});
  const {i18n} = useExtensionApi();

  const {
    deliveryPolicy,
    lines,
    orders,
    shippingAddress,
    priceBreakdownEstimate,
    status,
    upcomingBillingCycles,
  } = data.subscriptionContract;

  return (
    <Page
      title={i18n.translate('manageSubscription')}
      secondaryAction={<Button to="extension:/" />}
      primaryAction={
        <DetailsActions
          contractId={contractId}
          status={status}
          refetchSubscriptionContract={refetchSubscriptionContract}
        />
      }
    >
      <Grid columns={['fill', 'fill']} spacing="loose">
        <GridItem>
          <UpcomingOrderCard
            contractId={contractId}
            upcomingBillingCycles={upcomingBillingCycles}
          />
          <OverviewCard
            deliveryPolicy={deliveryPolicy}
            shippingAddress={shippingAddress}
          />
        </GridItem>
        <GridItem>
          <PriceSummaryCard
            price={priceBreakdownEstimate}
            lines={lines}
          />
          <PastOrdersCard orders={orders} />
        </GridItem>
      </Grid>
    </Page>
  );
}
```

## API Integration

The extension uses custom hooks for data fetching:

```typescript
export function useSubscriptionListData() {
  const {graphql} = useExtensionApi();
  
  const {data, loading, error, refetch} = graphql(
    SubscriptionListQuery,
    {variables: {}}
  );
  
  return {
    data: data?.customer,
    loading,
    error,
    refetchSubscriptionListData: refetch,
  };
}
```

## Routing

The extension implements client-side routing:

```tsx
export function Router() {
  const currentEntry = useNavigationCurrentEntry();
  const url = new URL(currentEntry.url);

  if (url.pathname.includes('subscriptions')) {
    const id = getSubscriptionIdFromPath(url.pathname);
    return <SubscriptionDetails id={id} />;
  }
  
  return <SubscriptionList />;
}
```
