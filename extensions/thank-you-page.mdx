The Thank You Page extension displays a subscription management link on the post-purchase order status page when an order contains subscription items.

## Extension Configuration

Configured in `shopify.extension.toml`:

```toml
api_version = "2025-10"

[[extensions]]
name = "Manage subscription link"
handle = "thank-you-page"
type = "ui_extension"

[[extensions.targeting]]
module = "./src/ThankYouPageManageSubscriptions.tsx"
target = "purchase.thank-you.block.render"

[extensions.capabilities]
api_access = true
```

### Extension Target

- **Target**: `purchase.thank-you.block.render`
- **Surface**: Order status / thank you page
- **Visibility**: Only shown when order contains subscription items

## Implementation

The extension uses Shopify's `@shopify/ui-extensions/preact` library:

```tsx
import '@shopify/ui-extensions/preact';
import {render} from 'preact';

export function ThankYouPageManageSubscriptions() {
  const shop = shopify.shop;
  const isEditor = shopify.extension.editor;
  const defaultAccountURL = `${shop?.storefrontUrl || ''}/account`;
  
  const hasSubscription =
    shopify.lines?.value?.some(
      (line) => line.merchandise?.sellingPlan?.recurringDeliveries,
    ) ?? false;
  
  // Hide if no subscription items (except in editor)
  if (!hasSubscription && !isEditor) {
    return null;
  }
  
  const translate = shopify.i18n.translate;
  
  return (
    <s-stack direction="block" gap="base">
      {isEditor ? (
        <s-banner tone="info">
          This extension is only visible for orders with subscriptions.
        </s-banner>
      ) : null}
      <s-box padding="base" border="base" borderRadius="base">
        <s-stack direction="block" gap="base">
          <s-heading>{translate('title')}</s-heading>
          <s-text>{translate('content')}</s-text>
          <s-button variant="secondary" href={defaultAccountURL}>
            {translate('manageSubscriptionLink')}
          </s-button>
        </s-stack>
      </s-box>
    </s-stack>
  );
}

export default async () => {
  render(<ThankYouPageManageSubscriptions />, document.body);
};
```

## Key Features

### Conditional Rendering

The extension only displays when the order contains subscription items:

```tsx
const hasSubscription =
  shopify.lines?.value?.some(
    (line) => line.merchandise?.sellingPlan?.recurringDeliveries,
  ) ?? false;

if (!hasSubscription && !isEditor) {
  return null;
}
```

### Editor Preview

When viewed in the theme editor, shows an informational banner:

```tsx
{isEditor ? (
  <s-banner tone="info">
    This extension is only visible for orders with subscriptions.
  </s-banner>
) : null}
```

### Account Link

Directs customers to their account page:

```tsx
const defaultAccountURL = `${shop?.storefrontUrl || ''}/account`;

<s-button variant="secondary" href={defaultAccountURL}>
  {translate('manageSubscriptionLink')}
</s-button>
```

- **Compatible**: Uses React-like API
- **Optimized**: Ideal for post-purchase extensions

## Troubleshooting

<AccordionGroup>
  <Accordion title="Extension not showing">
    Verify:
    - Order contains subscription items
    - Extension is enabled in theme settings
    - `purchase.thank-you.block.render` target is supported
  </Accordion>
  
  <Accordion title="Link not working">
    Check:
    - Storefront URL is correctly formatted
    - Customer account is enabled

  </Accordion>
  
  <Accordion title="Translations missing">
    Ensure:
    - Locale files exist in `/locales` directory
