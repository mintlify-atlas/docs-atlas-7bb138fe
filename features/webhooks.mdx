## Subscription Webhooks

### Contract Creation

Handles new subscription contract creation and sends welcome emails.

<CodeGroup>
```typescript Route Handler
// app/routes/webhooks.subscription_contracts.create.tsx
import {authenticate} from '~/shopify.server';
import {CustomerSendEmailJob, TagSubscriptionOrderJob, jobs} from '~/jobs';
import {CustomerEmailTemplateName} from '~/services/CustomerSendEmailService';
import {FIRST_ORDER_TAGS} from '~/jobs/tags/constants';

export const action = async ({request}: ActionFunctionArgs) => {
  const {topic, shop, payload} = await authenticate.webhook(request);

  logger.info({topic, shop, payload}, 'Received webhook');

  const {admin_graphql_api_origin_order_id: orderId} = payload;
  
  if (orderIsFromCheckout(orderId)) {
    jobs.enqueue(
      new CustomerSendEmailJob({
        shop,
        payload: {
          ...(payload as Webhooks.SubscriptionContractsCreate),
          emailTemplate: CustomerEmailTemplateName.NewSubscription,
        },
      })
    );
  }

  jobs.enqueue(
    new TagSubscriptionOrderJob({
      shop,
      payload: {
        orderId: payload.admin_graphql_api_origin_order_id,
        tags: FIRST_ORDER_TAGS,
      },
    })
  );

  return new Response();
};

function orderIsFromCheckout(orderId: string | null): boolean {
  return orderId !== null;
}
```

```typescript Webhook Payload
interface SubscriptionContractsCreate {
  admin_graphql_api_id: string;
  admin_graphql_api_origin_order_id: string | null;
  admin_graphql_api_customer_id: string;
  status: string;
  billing_policy: BillingPolicy;
  delivery_policy: DeliveryPolicy;
  lines: SubscriptionLine[];
}
```
</CodeGroup>

**Processing flow:**

<Steps>
  <Step title="Authenticate Webhook">
    Verify webhook authenticity via Shopify authentication
  </Step>
  
  <Step title="Check Origin Order">
    Determine if subscription was created from checkout
  </Step>
  
  <Step title="Send Welcome Email">
    Enqueue customer email job for new subscriptions
  </Step>
  
  <Step title="Tag Order">
    Add subscription tags to the origin order
  </Step>
</Steps>

## Billing Webhooks

### Billing Attempt Success

Handles successful billing attempts and stops dunning processes.

```typescript
// app/routes/webhooks.subscription_billing_attempts.success.tsx
import {DunningStopJob, TagSubscriptionOrderJob, jobs} from '~/jobs';
import {RECURRING_ORDER_TAGS} from '~/jobs/tags/constants';

export const action = async ({request}: ActionFunctionArgs) => {
  const {topic, shop, payload} = await authenticate.webhook(request);

  logger.info({topic, shop, payload}, 'Received webhook');

  jobs.enqueue(
    new DunningStopJob({
      shop,
      payload: payload as Webhooks.SubscriptionBillingAttemptSuccess,

        orderId: payload.admin_graphql_api_order_id,

  );
```

### Translation Generation Job

The `CreateSellingPlanTranslationsJob` generates localized selling plan descriptions:

```typescript
// app/jobs/webhooks/CreateSellingPlanTranslationsJob.ts
export class CreateSellingPlanTranslationsJob extends Job<
  Jobs.Parameters<Webhooks.SellingPlanGroups>
> {
  public queue: string = 'webhooks';

  async perform(): Promise<void> {
    const {shop, payload} = this.parameters;
    const {
      admin_graphql_api_id: sellingPlanGroupId,
      admin_graphql_api_app: appGid,
    } = payload;

    // Verify webhook is from subscriptions app
    if (appGid !== env.appGID) {
      this.logger.info(
        {subscriptionAppID: env.appGID, sourceAppID: appGid},
        'Not creating translation since event is not from subscriptions app',
      );
      return;
    }

    const {admin} = await unauthenticated.admin(shop);
    const {shopLocalesWithoutPrimary} = await getShopLocales(admin.graphql);
    const {shop: shopInfo} = await getShopInfos(admin.graphql);

    const response = await admin.graphql(SellingPlanTranslations, {
      variables: {id: sellingPlanGroupId},
    });

    const json = await response.json();
    const sellingPlanNodes = nodesFromEdges(
      json.data?.sellingPlanGroup?.sellingPlans.edges || [],
    );

    const sellingPlans = sellingPlanNodes.map((plan) => ({
      id: plan.id,
      pricingPolicies: plan.pricingPolicies as SellingPlanPricingPolicy[],
      deliveryPolicy: plan.deliveryPolicy as RecurringPolicy,
    }));

    await createSellingPlanTranslations(
      admin.graphql,
      sellingPlans,
      shopLocalesWithoutPrimary,
      shopInfo.currencyCode,
    );
  }
}
```

**Translation process:**

<Steps>
  <Step title="Verify App Source">
    Check that webhook originated from subscriptions app
  </Step>
  
  <Step title="Query Shop Locales">
    Fetch all shop locales excluding primary language
  </Step>
  
  <Step title="Fetch Selling Plans">
    Retrieve selling plan details including pricing and delivery policies
  </Step>
  
  <Step title="Generate Translations">
    Create localized descriptions for each selling plan in all shop locales
  </Step>
</Steps>

<Note>
Translations are generated for all shop locales to provide a localized checkout experience for international customers.
</Note>

## Error Handling

```typescript
export const action = async ({request}: ActionFunctionArgs) => {
  try {
    const {topic, shop, payload} = await authenticate.webhook(request);
    logger.info({topic, shop, payload}, 'Received webhook');

    // Process webhook
    await processWebhook(shop, payload);

    return new Response(null, {status: 200});
  } catch (error) {
    logger.error({error}, 'Failed to process webhook');
    
    // Return 500 to trigger Shopify retry
    return new Response(null, {status: 500});
  }
};
```

<Warning>
Returning a non-2xx status code will cause Shopify to retry the webhook delivery according to their retry policy.
</Warning>
