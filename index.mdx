## What's Included

This reference app provides everything you need to build a subscription application:

- **Complete Subscription Management** - Create and manage selling plan groups with flexible pricing and delivery options
- **Customer Experience** - Customer-facing UI extensions for managing subscriptions from their account
- **Admin Tools** - Shopify admin extensions for merchant subscription management
- **Billing Infrastructure** - Automated billing cycle processing with retry logic and dunning workflows
- **Job Scheduling** - Background job system for processing billing, sending emails, and managing inventory
- **Multi-locale Support** - Built-in translation management for international merchants
- **Production Ready** - Includes database schema, authentication, and deployment configuration

## Built With Modern Tools

The reference app leverages the latest Shopify development stack:

- **[Remix](https://remix.run/)** - Full-stack web framework for fast, resilient apps
- **[Shopify App Bridge](https://shopify.dev/docs/api/app-bridge)** - Seamless integration with the Shopify admin
- **[Polaris](https://polaris.shopify.com/)** - Shopify's design system for consistent UI
- **[Prisma](https://www.prisma.io/)** - Type-safe database access
- **[GraphQL](https://graphql.org/)** - Powerful API queries with Shopify's Admin API

## Getting Started

<Steps>
  <Step title="Clone the repository">
    Use the Shopify CLI to initialize a new app from this template:
    ```bash
    shopify app init --template https://github.com/Shopify/subscriptions-reference-app
    ```
  </Step>
  
  <Step title="Install dependencies">
    Install all required packages:
    ```bash
    pnpm install
    ```
  </Step>
  
  <Step title="Start development">
    Run the app in development mode:
    ```bash
    pnpm dev
    ```
  </Step>
</Steps>

<Note>
This is a reference implementation. If you're building for the Shopify App Store, ensure your app provides [substantive differences](https://shopify.dev/docs/apps/build/purchase-options/subscriptions/subscriptions-app#uniqueness-from-other-apps) from this reference app.
</Note>
